# Code of Conduct

This project aims to provide a welcoming, harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent; ask questions before making accusations.
- Focus on what is best for the community and the project.

## Unacceptable behavior

- Harassment, hate speech, or discriminatory language.
- Personal attacks, doxxing, or threats.
- Repeated disruption of discussions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and other contributions that do not align
with this Code of Conduct. For issues, open a GitHub issue with details and context.
